## 1. 개요

### 1.1 목적
현재 SVG 기반 시각화 지도를 구글 지도 실제 경로 캡처 이미지로 교체하여 정확한 이동 동선을 제공합니다.

### 1.2 배경
- 기존: 단순화된 SVG 시각화 (추상적)
- 개선: 구글 지도 실제 경로 스크린샷 (구체적)

---

## 2. 변경된 일정

| 일차 | 날짜 | 주요 일정 | 이동 경로 |
|------|------|----------|----------|
| **1일차** | 1/1 (수) | 덕산 참숯랜드 이용 후 숙박 | 출발지 → 덕산참숯랜드 → 숙소 |
| **2일차** | 1/2 (목) | 스플라스 리솜 워터파크 → 용현휴양림 | 숙소 → 스플라스 리솜 → 용현자연휴양림 |
| **3일차** | 1/3 (금) | 수덕사, 예당호 출렁다리 → 복귀 | 용현휴양림 → 수덕사 → 예당호 → 귀가 |

## 4. 기술 구현

### 4.1 이미지 파일 구조

```
public/
└── images/
    ├── route-day1.png   # 1일차 동선
    ├── route-day2.png   # 2일차 동선
    └── route-day3.png   # 3일차 동선
```

### 4.2 코드 수정 범위

| 파일 | 수정 내용 |
|------|----------|
| `src/App.jsx` | schedule 객체 날짜 업데이트 (1/1-1/3) |
| `src/App.jsx` | SVG 시각화 섹션 → 이미지 컴포넌트 교체 |
| `src/App.jsx` | route 경로 URL 업데이트 |

### 4.3 스크린샷 캡처 방법

```powershell
# Playwright를 사용한 구글 지도 캡처
npx playwright screenshot "https://www.google.com/maps/dir/..." public/images/route-day1.png
```
